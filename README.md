# Tactical Monster Game 🎮

A tactical turn-based strategy game developed using **C++** and the **Qt Framework**. This project focuses on implementing game mechanics, grid-based movement, and strategic monster combat in a GUI environment.

## 👥 Developers

* Khairullah Ebrahimi
* **Enayatullah Balaghi ([@EngEnayat](https://www.google.com/search?q=https://github.com/EngEnayat))**

## 🛠 Tech Stack

* **Language:** C++
* **Framework:** Qt (Qt Widgets / Qt Graphics View Framework)
* **IDE:** Qt Creator / CMake
* **Version Control:** Git

## 🚀 Key Features

* **Turn-Based Combat System:** Implemented core logic for player turns and action points.
* **Qt GUI Rendering:** Utilizing Qt's graphics framework for smooth rendering of the game board and sprites.
* **Strategy Mechanics:** Grid-based movement and unique monster abilities.
* **Event Handling:** Custom event management for user input and game states.

## 📋 Prerequisites

To build and run this project, you need:

* **Qt SDK (6.x recommended)**
* **Compiler supporting C++17 or later**
* **CMake (3.16+)**

## 🛠 How to Build & Run

1. **Clone the repository:**
```bash
git clone https://github.com/EngEnayat/Tactical-Monster-Game.git

```


2. **Open the project:**
* Open the `CMakeLists.txt` file in **Qt Creator**.


3. **Configure & Build:**
* Select your preferred Kit (Desktop Qt) and click the **Build** button.


4. **Run:**
* Execute the application from within Qt Creator or the build directory.
